# react-memo

Simple memory game

## Requirements

- `node` (`v14.15` recommended)
- `yarn` (`1.22.10` recommended)

## How to install

```sh
git clone git@github.com:niedzielnyaniol/react-memo.git

cd react-memo/

yarn install
```

## How to run

### Development env

```sh
yarn start
```

### Production env

```sh
yarn build

npx serve -s build
```

## Storybook

```sh
yarn storybook
```
